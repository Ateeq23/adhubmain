# Form Plugin Logos

This directory should contain logo images for the supported form plugins.

## Required Logos (16 total)

Place the following logo files in this directory:

1. **contact-form-7.png** - Contact Form 7 logo
2. **wpforms.png** - WPForms logo
3. **gravityforms.png** - Gravity Forms logo
4. **formidable.png** - Formidable Forms logo
5. **fluentform.png** - Fluent Forms logo
6. **ninjaforms.png** - Ninja Forms logo
7. **calderaforms.png** - Caldera Forms logo
8. **everestforms.png** - Everest Forms logo
9. **forminator.png** - Forminator logo
10. **happyforms.png** - HappyForms logo
11. **quform.png** - Quform logo
12. **superforms.png** - Super Forms logo
13. **metform.png** - MetForm logo
14. **weforms.png** - weForms logo
15. **divi.png** - Divi logo
16. **elementor-pro.png** - Elementor Pro logo

## Image Specifications

- **Format**: PNG or SVG (preferred)
- **Size**: 50x50 pixels or similar square aspect ratio
- **Background**: Transparent or white
- **Naming**: Use lowercase with hyphens (e.g., `contact-form-7.png`)

## Usage

The logos are used in the admin interface to identify each form plugin in the forms management section. If a logo is not found, a default WordPress forms icon will be displayed instead.
