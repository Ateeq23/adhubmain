Connect your WordPress forms to ADHUB lead management system for seamless lead capture and management.

== Description ==

ADHUB is a powerful WordPress plugin that integrates with popular form plugins to send form submissions directly to your ADHUB account. This enables seamless lead capture and management without any coding required.

### Features

* **Multi-Form Support**: Integrates with 16+ popular WordPress form plugins
* **Easy Setup**: Simple API key authentication
* **Form Management**: Enable/disable specific forms from the admin panel
* **Real-time Sync**: Form submissions are sent instantly to ADHUB
* **AJAX-powered**: No page reloads required for form management

### Supported Form Plugins

2. WPForms
3. Gravity Forms
4. Formidable Forms

### How It Works

1. Install and activate the ADHUB plugin
2. Enter your ADHUB API key in the settings
3. The plugin automatically detects your installed form plugins
4. Enable the forms you want to track
5. All form submissions will be sent to your ADHUB account

== Installation ==

1. Upload the `adhub` folder to the `/wp-content/plugins/` directory
2. Activate the ADHUB plugin through the 'Plugins' menu in WordPress
3. Navigate to the ADHUB settings page
4. Enter your ADHUB API key to authenticate
5. Enable the forms you want to track

== Frequently Asked Questions ==

= How do I get an ADHUB API key? =

Log in to your ADHUB account and navigate to the API settings to generate an API key.

= Do I need to install all 16 form plugins? =

No, the plugin only works with the form plugins you have installed and activated on your website.

= Can I enable/disable specific forms? =

Yes, you can enable or disable individual forms from the ADHUB admin panel.

= Is this plugin compatible with my theme? =

Yes, ADHUB works with any WordPress theme. It integrates at the form plugin level, not the theme level.
